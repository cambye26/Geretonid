Les méthodes HTTP utilisées

GET Récupérer une ressource ou une liste de ressources

PUT Mettre à jour une ressource
DELETE Supprimer une ressource

Les code http utilisés

200 Ok
201 Created
400 Bad Request
401 Unauthorized
403 Forbidden
404 Not Found

User

Les propriétés de User

  • id (Integer) : identifiant unique
  • email (String) : adresse e-mail | identifiant de connexion
  • password (String) : mot de passe
  • token (String) : jeton d'identification pour les requêtes
  • created_at (Datetime) : date de creation de l'ulilisateur
  • id (string) : identifiant unique

Post

Les propriétés de Post

  • id (Integer) : identifiant unique
  • title (String) : Titre de l'article
  • content (Text) : Contenu de l'article
  • excerpt (Text) : Extrait de l'article
  • poster (String) : Image de l'article
  • status (String) : Status de l'article
  • slug (String) : Slug de l'article
  • created_at (DateTime) : Date de création de l'article
  • updated_at (DateTime) : Date de la dernière modification de l'article

Requetes Post

Récupérer les posts
